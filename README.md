# Responsive Layouts — Flexbox & Grid Playground

This project demonstrates the difference between CSS Flexbox and CSS Grid by creating the same responsive 3-column pricing layout using both techniques.

## Project Versions

### Flexbox Version
- Built using CSS Flexbox.
- Uses flex-direction and alignment properties to arrange pricing cards.
- Best suited for arranging elements in one direction (row or column).

### Grid Version
- Built using CSS Grid.
- Uses grid-template-columns to create the pricing structure.
- Provides better control over rows and columns.

## Flexbox vs Grid Comparison

For this pricing layout, Flexbox was easier for placing the cards horizontally and managing spacing between items. It required fewer properties for a simple one-dimensional layout.

CSS Grid was easier for controlling the complete three-column structure because it is designed for two-dimensional layouts. However, Grid syntax felt slightly more complex at the beginning.

For responsive design, both methods worked well. Flexbox used flex-direction changes, while Grid used changing grid-template-columns to stack the cards into a single column.
